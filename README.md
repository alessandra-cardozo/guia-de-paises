

**Projeto: Guia de Países**  
Este projeto foi desenvolvido utilizando **HTML**, **CSS** e **JavaScript**.  
Ele permite que o usuário busque informações sobre um país específico, exibindo:  
- **Bandeira** do país;  
- **Capital**;
- **Continente**;  
- **População**;  
- **Moeda**;  
- **Língua oficial**.
